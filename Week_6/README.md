# Hyper Params Tuning

## 1. 파라미터 변동 및 고정 수치

- Learning Rate : 0.00001 고정
- epoch : 300 고정
- batch_Size : 16, 32, 128, 256 (현우)
- features : 3 ~ 10 (현우)
- n_critic : 1, 3, 7, 10 (준영)
- windowSize : 20, 30, 40, 50 (준영)
